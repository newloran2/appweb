Embedthis SQLite Library
===

This is custom build of the SQLite SQL engine.

### Prerequisites: 
    MakeMe (https://embedthis.com/me/download.html) for MakeMe to configure and build.

### To Build:

    ./configure
    me

    Alternatively to build without MakeMe:

    make
